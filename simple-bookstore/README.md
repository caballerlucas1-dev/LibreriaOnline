# LibroLoco - Simple Bookstore MVP

Un MVP básico para compra y venta de libros usando solo HTML, CSS y JavaScript vanilla.

## Cómo usar

1. Abre el archivo `index.html` en tu navegador web
2. Navega por los libros disponibles
3. Haz clic en "Comprar Ahora" para agregar libros al carrito
4. El indicador del carrito (círculo rojo arriba a la derecha) mostrará la cantidad

## Características

- Lista de libros predefinidos con portadas
- Diseño responsive que funciona en móvil y escritorio
- Simulación de agregar al carrito
- Animaciones y feedback visual simples
- Sin dependencias externas (solo HTML/CSS/JS puro)

## Estructura del proyecto

- `index.html` - Página principal
- `style.css` - Estilos básicos
- `books.js` - Datos de libros (hardcoded para el MVP)
- `script.js` - Lógica de la aplicación (renderizado, manejo de carrito, etc.)
